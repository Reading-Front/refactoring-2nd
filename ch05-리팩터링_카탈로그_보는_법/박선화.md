# 5장 리팩터링 카탈로그 보는 법

## 리팩터링 설명 형식

카탈로그의 리팩터링 기법들은 일정한 형식으로 정리했다. 각 기법에 대한 설명은 다섯 항목으로 구성된다.

1. **이름** - 가장 먼저 이름이 나온다. 이름은 리팩터링 용어를 구축하는 데 중요하다.
2. **개요** - 리팩터링의 핵심 개념을 표현한 개요(개념도 + 코드 예시)가 나온다.
   - 리팩터링 전후로 코드가 어떻게 달라지는지 보여준다.
3. **배경** - 해당 리팩터링 기법이 왜 필요한지와 그 기법을 적용하면 안되는 상황을 설명한다.
4. **절차** - 리팩터링하는 과정을 단계별로 제시한다.
   - 리팩터링을 안전하게 수행하려면 단계를 최대한 잘게 나누고 각 단계마다 테스트해야 한다.
5. **예시** - 해당 리팩터링 기법을 실제로 적용하는 예시와 효과를 보여준다.
